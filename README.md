# 🌐 عالم ماي باي | MYBY Universe

مرحباً بكم في المستودع الرسمي لتطبيق **MYBY Universe**، المنصة المتكاملة لخدمة المجتمع العربي في المملكة المتحدة.

## 🚀 عن المشروع
تطبيق **APP** هو المظلة الكبرى لجميع خدمات شركة **MYBY LTD**. يهدف المشروع إلى تقديم تجربة تسوق وحياة ذكية وشاملة من خلال واجهة واحدة سهلة الاستخدام.

## 🛠 الخدمات الحالية والمستقبلية:
* 🛒 **ماي باي جروسري:** تجربة تسوق مواد غذائية بريطانية وشرقية.
* 🏠 **الأدوات المنزلية:** تجهيزات عصرية لكل منزل.
* 👕 **الألبسة والمنسوجات:** أزياء مختارة بجودة عالية.
* 📚 **الكتب والإسلاميات:** غذاء الروح والمكتبة العربية.
* 🎉 **تنظيم المناسبات:** خدمات احترافية لجميع احتفالاتكم.
* 🤝 **خدمات المجتمع:** بوابة الدعم والترابط المجتمعي.

## 💻 التقنيات المستخدمة:
* **Frontend:** HTML5, CSS3 (Modern Responsive Design).
* **Backend:** Google Apps Script (GAS).
* **Database:** Google Sheets API.
* **Hosting:** Google Cloud Infrastructure.

## 📍 المقر الرئيسي
